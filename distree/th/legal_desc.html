ข้อตกลงในการเผยแพร่

ซอฟต์แวร์

Chantra เป็นโครงการของฝ่ายโอเพนซอร์ส ภายใต้สำนักงานส่งเสริมอุตสาหกรรมซอฟต์แวร์แห่งชาติ และพัฒนาโดยบริษัท Open Source Devlopment จำกัด Chantra เป็นการรวบรวมเอาซอฟต์แวร์โอเพนซอร์สที่เหมาะสมเข้าไว้ด้วยกันเพื่อความสะดวกในการเผยแพร่ ซอฟต์แวร์ทั้งหมดใน Chantra เป็นซอฟต์แวร์โอเพนซอร์ส ยกเว้น Java Runtime Environment ซึ่งถูกเผยแพร่โดยปราศจากการแก้ไขภายใต้ Binary Code License Agreement for the Java 2 Platform Standard Edition Runtime Environment 5.0 ของ Sun Microsystems, Inc. เพื่อใช้ในการทำงานของโปรแกรม OpenOffice.org และ FreeMind โครงการ Chantra และ soure code ของ Chantra สงวนลิขสิทธิ์ © 2548 โดยสำนักงานส่งเสริมอุตสาหกรรมซอฟต์แวร์แห่งชาติ และเผยแพร่ด้วยสัญญาอนุญาต GNU GPL

ทรัพย์สินทางปัญญา

ซอฟต์แวร์แต่ละตัวรวมทั้ง source code และเอกสารประกอบเป็นทรัพย์สินทางปัญญาของเจ้าของลิขสิทธิ์แต่ละราย ทั้งหมดได้รับการคุ้มครองด้วยกฎหมายลิขสิทธิ์ และเผยแพร่ด้วยสัญญาอนุญาตที่ได้อ้างถึงในคำอธิบายของซอฟต์แวร์แต่ละตัว ซอฟต์แวร์ทั้งหมดซึ่งเป็นซอฟต์แวร์โอเพนซอร์ส (นั่นคือยกเว้น Java Runtime Environment) อนุญาตให้คุณสามารถทำซ้ำ แก้ไข และเผยแพร่ซอฟต์แวร์ (ไม่ว่าจะแก้ไขหรือไม่) ต่อไปได้ ทั้งในรูปแบบ source code และ binary ทั้งนี้ขึ้นกับเงื่อนไขในสัญญาอนุญาตของซอฟต์แวร์แต่ละตัว

เครื่องหมายการค้าและ logo ที่ปรากฏใน Chantra เป็นสมบัติของเจ้าของเครื่องหมายการค้าและ logo เหล่านั้นแต่ละราย OpenOffice.org เป็นเครื่องหมายการค้าของ OpenOffice.org; Mozilla, Firefox, Thunderbird เป็นเครื่องหมายการค้าของ Mozilla Foundation; Audacity เป็นเครื่องหมายการค้าของ Dominic Mazzoni; Microsoft Windows, Office, Word, Excel และ PowerPoint เป็นเครื่องหมายการค้าหรือเครื่องหมายการค้าจดทะเบียนของ Microsoft Inc.; Sun, Sun Microsystems, StarOffice, Solaris และ Java เป็นเครื่องหมายการค้าหรือเครื่องหมายการค้าจดทะเบียนของ Sun Microsystems, Inc.; Linux เป็นเครื่องหมายการค้าจดทะเบียนของ Linus Torvalds; Mac OS X เป็นเครื่องหมายการค้าหรือเครื่องหมายการค้าจดทะเบียนของ Apple Computer, Inc.; Flash เป็นเครื่องหมายการค้าหรือเครื่องหมายการค้าจดทะเบียนของ Macromedia; ผลิตภัณฑ์ บริการ บริษัท หรือเครื่องหมายการค้าอื่น ๆ เป็นเครื่องหมายการค้าหรือเครื่องหมายการค้าจดทะเบียนของเจ้าของนั้น ๆ

ไม่มีการรับประกัน

Chantra เผยแพร่ซอฟต์แวร์ด้วยความตั้งใจที่จะให้เกิดประโยชน์ในวงกว้าง แต่เนื่องจากคุณได้รับอนุญาตให้ใช้โปรแกรมทั้งหมดได้โดยไม่คิดมูลค่า เหตุนี้จึงไม่มีการรับประกันใด ๆ ต่อโปรแกรม ไปจนถึงขอบเขตเท่าที่กฎหมายที่เกี่ยวข้องจะอำนวย เจ้าของลิขสิทธิ์และ/หรือบุคคลอื่นส่งมอบโปรแกรม "ตามลักษณะปัจจุบัน" โดยปราศจากการรับประกันในลักษณะใด ๆ ไม่ว่าโดยแจ้งชัดหรือโดยปริยาย รวมถึงแต่ไม่จำกัดเพียงการรับประกันโดยปริยายถึงความเหมาะสมในการวางตลาด หรือความเหมาะสมในวัตถุประสงค์โดยเฉพาะอย่างใดอย่างหนึ่ง ทั้งนี้เว้นแต่จะได้ระบุเอาไว้เป็นอย่างอื่นเป็นลายลักษณ์อักษร คุณเป็นผู้รับความเสี่ยงทั้งหมดในเรื่องคุณภาพหรือประสิทธิภาพของโปรแกรม หากปรากฏความจริงว่าโปรแกรมชำรุดบกพร่อง คุณจะเป็นผู้รับภาระต้นทุนในการบำรุงรักษา ซ่อมแซม หรือแก้ไขเท่าที่จำเป็นทั้งหมด

ไม่มีกรณีใด ๆ เว้นแต่ที่บังคับโดยกฎหมายที่เกี่ยวข้องหรือที่ตกลงกันไว้เป็นลายลักษณ์อักษร ที่เจ้าของลิขสิทธิ์หรือบุคคลอื่นใดผู้ซึ่งดัดแปลงและ/หรือเผยแพร่โปรแกรมต่อไปตามที่ได้กำหนดไว้ในสัญญาอนุญาตของซอฟต์แวร์แต่ละตัว จะต้องรับชดใช้ความเสียหายทั้งหลายต่อคุณ รวมถึงความเสียหายโดยทั่วไป โดยเฉพาะ โดยบังเอิญ หรือโดยเนื่องมาแต่เหตุ อันเป็นผลมาจากการใช้หรือความไม่สามารถในการใช้โปรแกรม (รวมถึงแต่ไม่จำกัดเพียงการสูญเสียของข้อมูล หรือการที่ข้อมูลถูกทำให้ผิดพลาด หรือความเสียหายอันคุณหรือบุคคลอื่นได้รับ หรือความล้มเหลวของโปรแกรมในการทำงานร่วมกับโปรแกรมอื่น) ถึงแม้ว่าเจ้าของลิขสิทธิ์หรือบุคคลอื่นดังกล่าวข้างต้นจะได้รับการแจ้งเตือนถึงความเป็นไปได้ของความเสียหายเช่นว่าแล้วก็ตาม

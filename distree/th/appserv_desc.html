AppServ

โปรแกรมติดตั้งที่รวมเอาซอฟต์แวร์โอเพนซอร์สคือ Apache, PHP, MySQL เอาไว้ใน package เดียว AppServ ทำให้สามารถติดตั้งโปรแกรมทั้งหมดในขั้นตอนเดียว ไม่ต้องติดตั้งและปรับแต่งทีละตัว AppServ จึงเหมาะสำหรับผู้ที่เขียนโปรแกรม PHP และใช้ MySQL ทำให้สามารถพัฒนาโปรแกรมเว็บบน Windows และเห็นผลลัพท์ได้ทันทีโดยไม่ต้องอัปโหลด AppServ ติดตั้งง่าย พร้อมทำงานได้ทันทีที่ติดตั้งเสร็จ ทั้งยังมี add-on ที่เมื่อติดตั้งแล้วทำให้คุณได้เว็บ Mambo, phpBB2 และ Moodle พร้อมใช้ได้ในทันที

เกี่ยวกับ Chantra
---------------------

Chantra เป็นแผ่นซีดีรวมซอฟต์แวร์ที่น่าใช้งานสำหรับวินโดวส์โดย SIPA เห็นว่าปัจจุบันมีซอฟต์แวร์โอเพนซอร์สจำนวนมหาศาล ซึ่งทำให้ผู้สนใจในการใช้งานโอเพนซอร์ส ค้นหาซอฟต์แวร์ที่ตนเองต้องการได้ยาก ทาง SIPA จึงคัดเลือกซอฟต์แวร์ที่โอเพนซอร์สที่มีคุณภาพรวบรวมอยู่ในซีดีเพื่อให้ง่าย ต่อการใช้งาน ซอฟต์แวร์ทั้งหมดในซีดี เป็นซอฟต์แวร์สำหรับระบบปฏิบัติการวินโดวส์ ซอฟต์แวร์ในซีดีส่วนมากจะมีเวอร์ชั่นที่ใช้บนลินุกซ์ได้ด้วยเพื่อทำให้ง่าย ขึ้น เมื่อต้องการเปลี่ยนไปใช้ลินุกซ์ในอนาคตทั้งนี้คุณสามารถติดตั้งโปรแกรมในซี ดีนี้โดยไม่ต้องลบโปรแกรมอื่นๆ ที่อยู่ในเครื่องออก 


โครงสร้างแผ่น Chantra
-----------------------------

/chrome		ข้อมูล xul application
/components	ไฟล์ components
/default		default config
/html		
  /en		คำอธิบายโปรแกรม
  /manuals	คู่มือการใช้งาน .pdf
  /programs	ไฟล์โปรแกรม
  /...
/xulrunner	xulrunner 1.9.0

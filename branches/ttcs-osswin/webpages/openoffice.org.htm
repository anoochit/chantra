OpenOffice.org Home Page

About OpenOffice.org 3.0.1

OpenOffice.org is an office productivity suite of programs (Writer, Calc, Impress, Draw, Base) which allows you to create/edit documents, spreadsheets, presentations, drawings and databases. It is compatible with all other major office suites.

Features of OpenOffice.org 3.0.1

Has a variety of third-party addons called extensions which offer additional functionality and/or features to OpenOffice.org
Edit files from other office suites
Reads and writes commonly used file formats, for example: Microsoft® Office files and WordPerfect files.
Export to PDF and .SWF (Flash)
OpenOffice.org has the built-in capability to export its files to the Adobe Acrobat® PDF and Macromedia® swf Flash file formats. Other supported formats include mobile device formats such as: AportisDoc (Palm), Pocket Word and Pocket Excel.
Support for OASIS OpenDocument format, an open XML-based file format
The OASIS OpenDocument format is a vendor and implementation independent file format

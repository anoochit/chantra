Firefox

เหตุผลสำคัญที่สุดที่คุณควรจะเปลี่ยนมาท่องเน็ตด้วย Firefox ก็คือ Firefox ช่วยให้คุณปลอดภัยจากการคุกคามของสปายแวร์ ที่ทำให้คุณเสียความเป็นส่วนตัวและท่องเน็ตได้ช้าลง Firefox ยังช่วยให้คุณท่องเน็ตได้สะดวกขึ้นด้วยการเปิดได้หลายเว็บพร้อมกันในคนละ tab และความสามารถที่แตกต่างอื่นๆ อีกมากมาย ถ้าความสามารถที่มากับ Firefox ยังไม่เพียงพอ คุณสามารถติดตั้ง extension อีกนับไม่ถ้วนที่เพิ่มความสามารถให้กับ Firefox ได้อย่างไม่จำกัด และ extension ที่คุณควรจะต้องติดตั้งคือ MozTLE ซึ่งทำให้ Firefox สามารถแบ่งบรรทัดภาษาไทยได้ถูกต้อง

Subversion

ซอฟต์แวร์ควบคุมเวอร์ชันของซอร์สโค้ด (version control) ที่ดีกว่า CVS ถ้าคุณต้องการพัฒนาซอฟต์แวร์ร่วมกันเป็นทีม และต้องการความสามารถในการเรียกกลับไฟล์เวอร์ชันเก่า Subversion เป็นคำตอบของคุณ เนื่องจาก Subversion เก็บเวอร์ชันทั้งไฟล์และ directory ทั้งการ copy, move และ rename ใน Subversion คุณสามารถ branch และ tag ได้อย่างไม่เปลืองทรัพยากร คุณสามารถใช้งาน Subversion ภายในเครื่องเดียว หรือใช้งานร่วมกันผ่านเน็ตเวิร์ก โดยใช้ standalone server, WebDav หรือผ่าน SSH
